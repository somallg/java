# jax-ws-mtom: Simple SOAP Web Service using JAX-WS (Java API for XML Web Services)
=======================================================================================

# What it is?
* This example demonstrates the use of *MTOM* (Message Transmission Optimization Mechanism) to optimize attachment size of a SOAP Message

# Build and Deploy
* mvn clean package
* Deploy war file to Tomcat or Jboss

# Run Client
* Run `PictureServiceClient`

* Run Test
* Run `SoapUI` with MTOM feature on and off to test the size of attachment
